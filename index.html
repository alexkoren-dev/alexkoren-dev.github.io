I handled a full cycle of development including graphic design, FE development using React, SEC API integration, ElasticSearch DB setup, backend development using Django, and hosting on AWS. TrashTalks

An Android app which promotes Recyling

A hackathon group project where we used the camera on a computer to represent the trashcan. Then we sent this picture to IBM Watson. Watson determines what is being recycled and calculates the number of points the items represent. Then Watson sends the information to ZXing (QR generator). We use an android app to scan the QR code. The App send the information to scoreboard/website (Heroku) via a database.
